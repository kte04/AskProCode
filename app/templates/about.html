About This Project

This project is designed to provide a platform for programmers to ask questions, share knowledge, and interact with each other. It aims to foster collaboration and learning in the coding community.

Features:

  • User registration and authentication
  • Create, edit, and delete posts
  • Commenting on posts
  • User roles (admin and user)
  • Tagging posts for better categorization

Technologies Used:

  • Flask
  • SQLAlchemy
  • Bootstrap
  • Flask-WTF (for forms)
  • Flask-Login (for authentication)
  • HTML/CSS
